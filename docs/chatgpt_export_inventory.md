## Mining Rules

- Do not import old ideas wholesale.
- Extract ideas into current docs only when they still fit current principles.
- Preserve older product ideas as "distant future" when they create social, legal, privacy, or economic risk.
- Prefer source references and summaries over copying long transcript passages.
- Treat personal-context transcripts carefully; mine design patterns, not private details.
